Write down your daily tasks.

## Features

- Add a task.
- List all tasks.
- Remove a task.
- All tasks are saved to disk in the temp folder.

## Running locally

Clone the project

```bash
  git clone https://github.com/Andre0n/Python-Todo-List.git
```

Change into the project directory

```bash
  cd Python-Todo-List
```

Run Linux/Mac

```bash
  python3 ./main.py
```

Run Windows

```bash
  py .\main.py
```
